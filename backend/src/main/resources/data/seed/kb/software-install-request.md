# How to Request Software Installation

All software installations on company devices must be approved and installed by IT. This policy ensures compliance, security, and license management.

## Before You Request

Check the **Software Catalog** first at [portal.company.com/software](https://portal.company.com/software). Many common applications are already available for self-service installation.

### Available in the Software Catalog
- Adobe Acrobat Reader
- Microsoft 365 Apps
- Visual Studio Code
- 7-Zip
- Zoom
- Notepad++

## Submitting a Request

### Step 1: Log In
Go to the **IT Service Portal** at [portal.company.com](https://portal.company.com) and log in with your company credentials.

### Step 2: New Request
Click **Request Software** from the dashboard or navigate to **IT Requests > Software Installation**.

### Step 3: Fill Out the Form
Provide the following information:
- **Software name and version**
- **Reason for request** (be specific — e.g., "Required for frontend development project")
- **Urgency**: Low / Medium / High
- **Approver** (your manager's email)

### Step 4: Approvals
1. Your **manager** receives an approval email
2. Once approved, **IT reviews** the request for licensing and security
3. If approved, IT installs the software remotely within **2 business days**

## Self-Service Installation
For approved software titles, you can install directly:
1. Open **Company Software Center** from your Start Menu
2. Browse or search for the application
3. Click **Install**
4. No admin credentials required — installation runs automatically

## FAQ

**Q: Can I install free software myself?**
A: No. All software must go through the approval process, even free tools. This ensures no license conflicts or security issues.

**Q: What about open-source tools?**
A: Open-source tools follow the same process. IT will verify the tool is safe and maintain an inventory.

**Q: How long does it take?**
A: Self-service installations are instant. Manual requests are processed within 2 business days after manager approval.
