# Troubleshooting Printer Issues

Follow these steps in order to resolve common printer problems.

## Printer is Offline

### Step 1: Check Physical Connections
- Ensure the printer is **powered on** (check the power light)
- Verify the **USB or Ethernet cable** is securely connected
- Check that the **network cable** is plugged in (for network printers)

### Step 2: Check Printer Status on Your Computer

**Windows:**
1. Go to **Settings > Bluetooth & Devices > Printers & Scanners**
2. Select your printer and click **Open print queue**
3. Ensure the status shows **Ready** (not "Offline" or "Paused")
4. If offline, click **Printer > Use Printer Online**

**macOS:**
1. Go to **System Settings > Printers & Scanners**
2. Select your printer — status should show **Idle**
3. If paused or stopped, click **Resume**

### Step 3: Restart the Print Spooler (Windows)
1. Open **Run** (`Win + R`), type `services.msc`, press Enter
2. Find **Print Spooler** in the list
3. Right-click and select **Restart**
4. Try printing again

## Paper Jam

1. **Open all printer access doors** — paper can get stuck in multiple places
2. **Gently remove any visible paper** — pull in the direction of paper feed, not backward
3. Check the **rear access panel** and **duplex unit** for jammed paper fragments
4. Close all doors and press the **OK** or **Resume** button

## Poor Print Quality

| Symptom | Likely Cause | Solution |
|---|---|---|
| Faded text | Low toner | Replace toner cartridge |
| Vertical streaks | Dirty drum/roller | Run printer cleaning cycle |
| Blank pages | Empty toner | Replace toner cartridge |
| Smudged ink | Wrong paper type | Use recommended paper stock |
| Lines/missing text | Clogged printhead | Run printhead cleaning (inkjet) |

## Printing Is Slow
- **Large files** take time to spool — reduce image resolution or print in draft mode
- **Network traffic** can slow network printers — try printing during off-peak hours
- Reduce print resolution from **Best** to **Normal** for everyday documents
