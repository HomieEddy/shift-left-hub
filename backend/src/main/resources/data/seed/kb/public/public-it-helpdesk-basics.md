# IT Helpdesk Basics

## What We Support

The IT Helpdesk provides support for the following:

| Category | Examples |
|----------|----------|
| Hardware | Laptops, monitors, printers, phones, peripherals |
| Software | Microsoft 365, company applications, VPN, antivirus |
| Network | Wi-Fi connectivity, VPN access, network drives |
| Accounts | Password resets, account unlocks, MFA issues |
| Access | File share permissions, application access, VPN |

## What We Do Not Support

- Personal devices not enrolled in company management
- Personal software or home network configuration
- Non-company email accounts or cloud services

## Submitting a Ticket

The best way to get help is through the **IT Service Portal** at https://helpdesk.company.com:

1. Log in with your corporate credentials
2. Click **Create Ticket**
3. Select the appropriate category:
   - **Incident** — something is broken or not working
   - **Service Request** — you need something new (hardware, software, access)
   - **Security Incident** — phishing, suspicious activity, data breach
4. Describe the issue with as much detail as possible
5. Include relevant screenshots or error messages
6. Click **Submit**

## Response Times

| Priority | Definition | Initial Response |
|----------|-----------|-----------------|
| Critical | System down, security incident, widespread outage | 15 minutes |
| High | Individual unable to work, major functionality blocked | 1 hour |
| Medium | Non-urgent issue, minor functionality impacted | 4 hours |
| Low | Questions, requests, cosmetic issues | 1 business day |

## Self-Service Options

Before submitting a ticket, check these self-service options:

- **Password Reset:** https://password.company.com
- **Software Installation:** https://software.company.com
- **Knowledge Base:** Search articles on the Intranet
- **FAQ:** Check the General FAQ in the Public workspace

## Tips for Faster Resolution

- Provide clear, detailed descriptions of the problem
- Include error messages and screenshots
- Specify which device and operating system you're using
- Note any recent changes that may have triggered the issue
- Keep your ticket number for follow-up reference

# Notions de base du service d'assistance informatique

## Ce que nous supportons

Le service d'assistance informatique offre du soutien pour les éléments suivants :

| Catégorie | Exemples |
|-----------|----------|
| Matériel | Ordinateurs portables, moniteurs, imprimantes, téléphones, périphériques |
| Logiciel | Microsoft 365, applications d'entreprise, VPN, antivirus |
| Réseau | Connectivité Wi-Fi, accès VPN, lecteurs réseau |
| Comptes | Réinitialisations de mots de passe, déverrouillages de comptes, problèmes MFA |
| Accès | Permissions de partage de fichiers, accès aux applications, VPN |

## Ce que nous ne supportons pas

- Appareils personnels non inscrits à la gestion d'entreprise
- Logiciels personnels ou configuration de réseau domestique
- Comptes de courriel non professionnels ou services cloud

## Soumettre un ticket

La meilleure façon d'obtenir de l'aide est via le **Portail de services informatiques** à https://helpdesk.company.com :

1. Connectez-vous avec vos identifiants professionnels
2. Cliquez sur **Créer un ticket**
3. Sélectionnez la catégorie appropriée :
   - **Incident** — quelque chose est cassé ou ne fonctionne pas
   - **Demande de service** — vous avez besoin de quelque chose de nouveau (matériel, logiciel, accès)
   - **Incident de sécurité** — hameçonnage, activité suspecte, violation de données
4. Décrivez le problème avec autant de détails que possible
5. Incluez des captures d'écran ou des messages d'erreur pertinents
6. Cliquez sur **Soumettre**

## Délais de réponse

| Priorité | Définition | Réponse initiale |
|----------|-----------|-----------------|
| Critique | Système en panne, incident de sécurité, panne généralisée | 15 minutes |
| Élevée | Employé incapable de travailler, fonctionnalité majeure bloquée | 1 heure |
| Moyenne | Problème non urgent, fonctionnalité mineure affectée | 4 heures |
| Faible | Questions, demandes, problèmes esthétiques | 1 jour ouvrable |

## Options en libre-service

Avant de soumettre un ticket, vérifiez ces options en libre-service :

- **Réinitialisation de mot de passe :** https://password.company.com
- **Installation de logiciel :** https://software.company.com
- **Base de connaissances :** Recherchez des articles sur l'Intranet
- **FAQ :** Consultez la FAQ générale dans l'espace de travail Public

## Conseils pour une résolution plus rapide

- Fournissez des descriptions claires et détaillées du problème
- Incluez les messages d'erreur et les captures d'écran
- Spécifiez l'appareil et le système d'exploitation que vous utilisez
- Notez tout changement récent qui pourrait avoir déclenché le problème
- Conservez votre numéro de ticket pour référence de suivi
