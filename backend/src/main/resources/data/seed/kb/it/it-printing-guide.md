# Printing Guide

## Printer Locations

Network printers are available on every floor. Each printer is labeled with its name and IP address.

| Location | Printer Name | Model | Features |
|----------|-------------|-------|----------|
| Floor 1 - Lobby | HP-MFP-Lobby | HP LaserJet Pro M404 | Print, Scan, Copy |
| Floor 2 - East | Canon-iR-East | Canon imageRUNNER 2625i | Print, Scan, Copy, Fax |
| Floor 2 - West | Canon-iR-West | Canon imageRUNNER 2625i | Print, Scan, Copy, Fax |
| Floor 3 - Open Area | HP-MFP-3rd | HP LaserJet Pro M404 | Print, Scan, Copy |

## Installing Printers

### Windows 11 (Automatic via Group Policy)

Company-issued laptops have printers deployed automatically via Group Policy. Printers appear in your devices list within 15 minutes of connecting to the office network.

### Windows 11 (Manual)

1. Open **Settings** → **Bluetooth & devices** → **Printers & scanners**
2. Click **Add device**
3. Select the printer from the list, or click **Add manually**
4. Enter the printer's IP address (available from the printer's display panel)
5. Follow the driver installation prompts

### macOS

1. Open **System Settings** → **Printers & Scanners**
2. Click **Add Printer, Scanner, or Fax**
3. The printer should appear automatically on the network
4. Select it and click **Add**

## Print Release (Follow-Me Printing)

All printers use secure print release to protect confidential documents:

1. Send your print job as usual (select any network printer)
2. Walk to any company printer
3. Tap your badge on the card reader or enter your employee ID
4. Select your print job from the queue
5. Tap **Print** to release

## Scanning

To scan a document to your email:

1. Place your document on the scanner glass or in the document feeder
2. On the printer touchscreen, select **Scan**
3. Choose **Scan to Email**
4. Verify your email address is displayed
5. Select color mode and resolution (300 DPI recommended for documents)
6. Press **Start**

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Printer not found | Verify you are connected to the office network (not VPN) |
| Print job stuck | Open the print queue → right-click the job → **Cancel** |
| Paper jam | Open the printer cover and follow the diagram inside to remove jammed paper |
| Toner low | Contact the facilities team to request a toner replacement |
| Scan not arriving | Check spam folder; verify email address on the printer display |

# Guide d'impression

## Installation des imprimantes

### Windows 11 (Automatique via la stratégie de groupe)

Les ordinateurs portables de l'entreprise ont des imprimantes déployées automatiquement via la stratégie de groupe. Les imprimantes apparaissent dans votre liste de périphériques dans les 15 minutes suivant la connexion au réseau du bureau.

### Windows 11 (Manuel)

1. Ouvrez **Paramètres** → **Bluetooth et périphériques** → **Imprimantes et scanners**
2. Cliquez sur **Ajouter un périphérique**
3. Sélectionnez l'imprimante dans la liste, ou cliquez sur **Ajouter manuellement**
4. Entrez l'adresse IP de l'imprimante (disponible sur l'écran de l'imprimante)
5. Suivez les invites d'installation du pilote

### macOS

1. Ouvrez **Réglages Système** → **Imprimantes et scanners**
2. Cliquez sur **Ajouter une imprimante, un scanner ou un télécopieur**
3. L'imprimante devrait apparaître automatiquement sur le réseau
4. Sélectionnez-la et cliquez sur **Ajouter**

## Libération d'impression (Impression suivie)

Toutes les imprimantes utilisent la libération d'impression sécurisée pour protéger les documents confidentiels :

1. Envoyez votre travail d'impression comme d'habitude (sélectionnez n'importe quelle imprimante réseau)
2. Rendez-vous à n'importe quelle imprimante de l'entreprise
3. Tapotez votre badge sur le lecteur de carte ou entrez votre identifiant d'employé
4. Sélectionnez votre travail d'impression dans la file d'attente
5. Appuyez sur **Imprimer** pour libérer

## Numérisation

Pour numériser un document vers votre courriel :

1. Placez votre document sur la vitre du scanner ou dans le chargeur de documents
2. Sur l'écran tactile de l'imprimante, sélectionnez **Numériser**
3. Choisissez **Numériser vers courriel**
4. Vérifiez que votre adresse courriel est affichée
5. Sélectionnez le mode couleur et la résolution (300 DPI recommandé pour les documents)
6. Appuyez sur **Démarrer**

## Dépannage

| Problème | Solution |
|----------|----------|
| Imprimante introuvable | Vérifiez que vous êtes connecté au réseau du bureau (pas au VPN) |
| Travail d'impression bloqué | Ouvrez la file d'impression → clic droit sur le travail → **Annuler** |
| Bourrage papier | Ouvrez le couvercle de l'imprimante et suivez le schéma à l'intérieur pour retirer le papier coincé |
| Toner faible | Contactez l'équipe des installations pour demander un remplacement de toner |
| Numérisation non reçue | Vérifiez le dossier spam ; vérifiez l'adresse courriel sur l'écran de l'imprimante |
