# Hardware Requests & Asset Management

## Hardware Request Process

All hardware requests must be submitted through the IT Service Portal. Standard processing times:

| Request Type | Typical Processing Time | Approval Required |
|-------------|------------------------|-------------------|
| New laptop (hire) | 3 business days | Manager |
| Laptop replacement | 5 business days | IT + Manager |
| Additional monitor | 2 business days | Manager |
| Keyboard / Mouse | 1 business day | Self-service |
| Headset | 1 business day | Self-service |
| Docking station | 3 business days | Manager |
| Mobile phone | 5 business days | VP-level |

## Standard Hardware Configuration

### Laptop Specifications

| Component | Standard Configuration |
|-----------|----------------------|
| Model | Dell Latitude 5540 |
| Processor | Intel Core i7-1355U |

### Monitor Specifications

| Component | Standard Configuration |
|-----------|----------------------|
| Model | Dell UltraSharp U2723QE |

## Submitting a Hardware Request

1. Log in to the IT Service Portal (https://helpdesk.company.com)
2. Select **Hardware Request** from the category menu
3. Choose the hardware type from the dropdown
4. Complete all required fields (justification, delivery location, cost code)
5. Attach manager approval if required
6. Submit the request

## Asset Tagging

All company hardware is tagged with an asset barcode:

- **Laptops** — tag on the bottom panel
- **Monitors** — tag on the back of the stand
- **Peripherals** — tag on the underside
- **Phones** — tag inside the battery compartment or on the back

Report any missing or damaged tags to IT.

## Equipment Return Procedures

When leaving the company or transferring departments:

1. Schedule a return appointment with IT via the Service Portal
2. Back up all personal files before the appointment
3. Sign out of all corporate and personal accounts
4. Return all hardware in its original packaging if available
5. IT will perform a data wipe and asset check
6. You will receive a confirmation of return via email

# Demandes de matériel et gestion des actifs

## Processus de demande de matériel

Toutes les demandes de matériel doivent être soumises via le Portail de services informatiques. Délais de traitement standards :

| Type de demande | Délai de traitement typique | Approbation requise |
|----------------|----------------------------|---------------------|
| Nouvel ordinateur portable (embauche) | 3 jours ouvrables | Gestionnaire |
| Remplacement d'ordinateur portable | 5 jours ouvrables | IT + Gestionnaire |
| Moniteur supplémentaire | 2 jours ouvrables | Gestionnaire |
| Clavier / Souris | 1 jour ouvrable | Libre-service |
| Casque | 1 jour ouvrable | Libre-service |
| Station d'accueil | 3 jours ouvrables | Gestionnaire |
| Téléphone mobile | 5 jours ouvrables | Niveau VP |

## Soumission d'une demande de matériel

1. Connectez-vous au Portail de services informatiques (https://helpdesk.company.com)
2. Sélectionnez **Demande de matériel** dans le menu des catégories
3. Choisissez le type de matériel dans la liste déroulante
4. Remplissez tous les champs obligatoires (justification, lieu de livraison, code de coût)
5. Joignez l'approbation du gestionnaire si nécessaire
6. Soumettez la demande

## Étiquetage des actifs

Tous les équipements de l'entreprise sont étiquetés avec un code-barres d'actif :

- **Ordinateurs portables** — étiquette sur le panneau inférieur
- **Moniteurs** — étiquette à l'arrière du support
- **Périphériques** — étiquette sur la face inférieure
- **Téléphones** — étiquette à l'intérieur du compartiment batterie ou à l'arrière

Signalez toute étiquette manquante ou endommagée au service informatique.

## Procédures de retour d'équipement

Lorsque vous quittez l'entreprise ou changez de service :

1. Planifiez un rendez-vous de retour avec le service informatique via le Portail de services
2. Sauvegardez tous vos fichiers personnels avant le rendez-vous
3. Déconnectez-vous de tous les comptes professionnels et personnels
4. Retournez tout le matériel dans son emballage d'origine si disponible
5. Le service informatique effectuera un effacement des données et une vérification des actifs
6. Vous recevrez une confirmation de retour par courriel
