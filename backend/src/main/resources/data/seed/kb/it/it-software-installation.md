# Software Installation Requests

## Approved Software List

The following software is pre-approved and available for self-service installation:

| Category | Software | License Type |
|----------|----------|-------------|
| Office Suite | Microsoft 365 Apps | Enterprise |
| Web Browsers | Chrome, Firefox, Edge | Free / Enterprise |
| Development | VS Code, IntelliJ Community, Git | Free |
| Communication | Slack, Microsoft Teams | Enterprise |
| PDF Tools | Adobe Acrobat Reader | Free |
| Antivirus | Defender for Business | Enterprise |
| Utilities | 7-Zip, Notepad++, Greenshot | Free |

## Self-Service Installation via Company Portal

1. Open the **Company Software Portal** (https://software.company.com)
2. Log in with your corporate credentials
3. Browse or search for the software you need
4. Click **Install**
5. The installation will begin automatically on your company-managed device
6. You will receive a notification when installation is complete

## Requesting Unlisted Software

If the software you need is not on the approved list:

1. Open the IT Service Portal (https://helpdesk.company.com)
2. Select **Software Request** as the category
3. Complete the form with:
   - Software name and version
   - Publisher/developer
   - Business justification
   - License information if you have it
4. The request will be reviewed by IT Security (typically within 2 business days)
5. You will receive an email with the approval decision

## Software Installation Policy

- **Do not** install unapproved software on company devices
- **Do not** use cracked or pirated software
- Software licenses must be assigned before installation
- All software must be kept up to date with security patches
- Personal software requires manager approval and may not be supported by IT
- Administrative privileges are not granted on company-managed devices

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Installation fails | Verify your device is connected to the company network or VPN |
| Software not found | Search by publisher name; request via Service Portal if not listed |
| License error | Contact the IT helpdesk to verify license assignment |
| Installation too slow | Install outside peak hours (9 AM - 11 AM) |

# Demandes d'installation de logiciels

## Liste des logiciels approuvés

Les logiciels suivants sont pré-approuvés et disponibles pour une installation en libre-service :

| Catégorie | Logiciel | Type de licence |
|-----------|----------|-----------------|
| Suite bureautique | Microsoft 365 Apps | Entreprise |
| Navigateurs Web | Chrome, Firefox, Edge | Gratuit / Entreprise |
| Développement | VS Code, IntelliJ Community, Git | Gratuit |
| Communication | Slack, Microsoft Teams | Entreprise |
| Outils PDF | Adobe Acrobat Reader | Gratuit |
| Antivirus | Defender for Business | Entreprise |
| Utilitaires | 7-Zip, Notepad++, Greenshot | Gratuit |

## Installation en libre-service via le Portail d'entreprise

1. Ouvrez le **Portail logiciel de l'entreprise** (https://software.company.com)
2. Connectez-vous avec vos identifiants professionnels
3. Parcourez ou recherchez le logiciel dont vous avez besoin
4. Cliquez sur **Installer**
5. L'installation commencera automatiquement sur votre appareil géré par l'entreprise
6. Vous recevrez une notification lorsque l'installation sera terminée

## Demande de logiciels non répertoriés

Si le logiciel dont vous avez besoin ne figure pas sur la liste approuvée :

1. Ouvrez le Portail de services informatiques (https://helpdesk.company.com)
2. Sélectionnez **Demande de logiciel** comme catégorie
3. Remplissez le formulaire avec :
   - Nom et version du logiciel
   - Éditeur/développeur
   - Justification commerciale
   - Informations sur la licence si vous en disposez
4. La demande sera examinée par la sécurité informatique (généralement sous 2 jours ouvrables)
5. Vous recevrez un courriel avec la décision d'approbation

## Politique d'installation de logiciels

- **N'installez pas** de logiciels non approuvés sur les appareils de l'entreprise
- **N'utilisez pas** de logiciels piratés ou crackés
- Les licences logicielles doivent être attribuées avant l'installation
- Tous les logiciels doivent être maintenus à jour avec les correctifs de sécurité
- Les logiciels personnels nécessitent l'approbation du gestionnaire et peuvent ne pas être pris en charge par le service informatique
- Les privilèges d'administrateur ne sont pas accordés sur les appareils gérés par l'entreprise

## Dépannage

| Problème | Solution |
|----------|----------|
| L'installation échoue | Vérifiez que votre appareil est connecté au réseau de l'entreprise ou au VPN |
| Logiciel introuvable | Recherchez par nom d'éditeur ; faites une demande via le Portail de services si non répertorié |
| Erreur de licence | Contactez le service d'assistance informatique pour vérifier l'attribution de la licence |
| Installation trop lente | Installez en dehors des heures de pointe (9 h - 11 h) |
