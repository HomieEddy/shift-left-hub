# Milestone v1.0: Initial MVP

**Status:** ✅ SHIPPED 2026-06-08

## Overview

Initial release of the Shift-Left Knowledge Hub — a modern ITSM portal with conversational AI triage, contextual ticketing, agent dashboard, and KCS auto-drafting. Full CI/CD pipeline with production-quality test coverage.

## Milestone Summary

**Key Decisions:**
- Modular Monolith over microservices — solo dev, no distributed complexity
- PostgreSQL FTS + pgvector — no extra infrastructure
- JWT with HttpOnly cookies — security best practice
- Bilingual EN/FR from day one — rare differentiator
- Spring AI for KCS drafting — native ecosystem integration
- JaCoCo + SonarQube Cloud for quality gating

**Issues Resolved:**
- All integration tests passing with Testcontainers + pgvector
- CI pipeline operational with branch protection
- SonarCloud quality gate passing (security hotspots resolved)
- 16 ESLint errors fixed across frontend

**Technical Debt Incurred:**
- Some phase directories lack SUMMARY.md files (documentation gap)
- Phase 4 escalation data layer and Phase 7 quality documentation not formalized as plans
