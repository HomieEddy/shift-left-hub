<interfaces>
Plan 03-01 created these API endpoints that this plan consumes:

GET /api/ai/config → AiConfigResponse {
  llmProvider: string,
  ollamaEndpointUrl: string,
  hasOpenaiKey: boolean,
  chatModelName: string,
  embeddingModelName: string,
  similarityThreshold: number,
  embeddingDimension: number
}

PUT /api/ai/config { body: AiConfigRequest } → AiConfigResponse

POST /api/ai/config/test { body: AiConfigRequest } → TestConnectionResult {
  success: boolean,
  message: string
}

POST /api/ai/config/embeddings/reindex → { message: string }

All endpoints require authentication (HttpOnly cookie via withCredentials: true).
PUT and POST test/embeddings/reindex require ADMIN role.

Existing Angular patterns:
- Standalone components with `inject()` DI (no constructor injection)
- Lazy routes with `loadComponent`
- AuthService: `isAuthenticated()`, `isAdmin()` signals
- HttpClient with `withCredentials: true` pattern
- i18n: `translate('admin.users.title')` pipe/function pattern — check existing usage
- NgIf, NgFor from @angular/common
</interfaces>

    Define TypeScript interfaces for the API contracts:
    ```typescript
    export interface AiConfigResponse {
      llmProvider: string;
      ollamaEndpointUrl: string;
      hasOpenaiKey: boolean;
      chatModelName: string;
      embeddingModelName: string;
      similarityThreshold: number;
      embeddingDimension: number;
    }

    export interface AiConfigRequest {
      llmProvider?: string;
      ollamaEndpointUrl?: string;
      openaiApiKey?: string;
      chatModelName?: string;
      embeddingModelName?: string;
      similarityThreshold?: number;
    }

    export interface TestConnectionResult {
      success: boolean;
      message: string;
    }
    ```

    Service methods:
    - `getConfig(): Observable&lt;AiConfigResponse&gt;` → GET /api/ai/config
    - `updateConfig(config: AiConfigRequest): Observable&lt;AiConfigResponse&gt;` → PUT /api/ai/config
    - `testConnection(config: AiConfigRequest): Observable&lt;TestConnectionResult&gt;` → POST /api/ai/config/test
    - `reindexEmbeddings(): Observable&lt;{ message: string }&gt;` → POST /api/ai/config/embeddings/reindex

    Use `@Injectable({ providedIn: 'root' })` with `private http = inject(HttpClient)`.
    All requests use `{ withCredentials: true }` per existing auth pattern.
