    **12. Rate limiting on auth endpoints (MEDIUM priority)**
    Auth endpoints (`/api/auth/**`) are publicly accessible (permitAll) and have no brute-force protection.
    
    Fix approach:
    a. Add a `@Component` class `RateLimitingFilter` in the config package that:
       - Tracks request counts per IP address for `/api/auth/*` endpoints
       - Uses a Caffeine cache with 1-minute expiry for counters
       - Limits to 10 requests per minute per IP on auth endpoints
       - Returns 429 Too Many Requests with Retry-After header when limit exceeded
       - Uses `OncePerRequestFilter` and is registered BEFORE the auth endpoints in SecurityConfig
    
    b. Add the filter to the SecurityConfig filter chain:
       Register `rateLimitingFilter()` bean and add it: `.addFilterBefore(rateLimitingFilter(), JwtAuthenticationFilter.class)`
       But ensure it only applies to `/api/auth/**` paths (check the request URI in the filter).
    
    Implementation:
    ```java
    @Component
    public class RateLimitingFilter extends OncePerRequestFilter {
        private final Cache<String, Integer> requestCounts;
        
        public RateLimitingFilter() {
            this.requestCounts = Caffeine.newBuilder()
                .expireAfterWrite(1, TimeUnit.MINUTES)
                .maximumSize(1000)
                .build();
        }
        
        @Override
        protected void doFilterInternal(HttpServletRequest request, HttpServletResponse response, FilterChain chain)
                throws ServletException, IOException {
            String path = request.getRequestURI();
            if (path.startsWith("/api/auth/")) {
                String clientIp = request.getRemoteAddr();
                Integer count = requestCounts.get(clientIp, k -> 0);
                if (count >= 10) {
                    response.setStatus(429);
                    response.setHeader("Retry-After", "60");
                    response.getWriter().write("{\"error\":\"Too many requests — please try again in 60 seconds\"}");
                    return;
                }
                requestCounts.put(clientIp, count + 1);
            }
            chain.doFilter(request, response);
        }
    }
    ```
