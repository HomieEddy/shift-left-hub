Key service interfaces (read before implementing):

AdminUserService methods:
- listUsers(): List<UserResponse>
- getUserByEmail(email): UserResponse
- updateUserRole(userId, role): UserResponse
- toggleUserEnabled(userId, enabled): UserResponse
- Prevents self-role-change (admin cannot demote themselves)

WorkspaceInvitationService methods:
- createInvitation(workspaceId, email, role): InvitationResponse
- acceptInvitation(token): void
- rejectInvitation(token): void
- getPendingInvitations(workspaceId): List<InvitationResponse>
- Invitations expire after 7 days (checks expiry on accept)

AiChatService methods:
- chat(workspaceId, request): Flux<ChatResponse> (SSE streaming)
- Uses ChatClient from Spring AI, fetches workspace config
- Builds prompt with context from hybrid search results
- Never stores raw API responses

EmbeddingService methods:
- embed(text): float[] vector
- embedBatch(texts): List<float[]>
- Uses configured embedding model from workspace config

PublicArticleService methods:
- searchPublished(workspaceId, query, tags, pageable): Page<ArticleResponse>
- getPublishedArticle(workspaceId, articleId): ArticleResponse
- Only returns PUBLISHED articles
- Filters by workspace isolation

AiConfigService methods:
- getConfig(workspaceId): AiConfig
- encrypt(apiKey): String (AES/GCM)
- decrypt(encryptedKey): String
- testConnection(provider, endpoint, apiKey, model): ConnectionTestResult
