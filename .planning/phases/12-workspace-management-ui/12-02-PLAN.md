  <files>

  <action>
    Add to `WorkspaceService.java`:

    1. **updateWorkspace(UUID id, String name, String description, String icon)**:
       - Find workspace by id (throw RuntimeException if not found)
       - Update name if non-null, description if non-null, icon if non-null
       - Save and return
       - Do NOT allow updating the Default Workspace's name (check slug == "default")

    2. **softDeleteWorkspace(UUID id)**:
       - Find workspace by id (throw RuntimeException if not found)
       - If workspace slug == "default", throw IllegalArgumentException("Cannot delete the Default Workspace")
       - Set deletedAt = LocalDateTime.now()
       - Save and return

    3. **leaveWorkspace(UUID workspaceId, UUID userId)**:
       - If workspace slug == "default", throw IllegalArgumentException("Cannot leave the Default Workspace")
       - Get the member record
       - Check if user is the only admin (D-18): count ADMINS in workspace → if 1 AND user is admin, throw IllegalArgumentException
       - Delete the WorkspaceMember record
       - Log the leave event

    4. **removeMember(UUID workspaceId, UUID userId)**:
       - Same only-admin guard as leaveWorkspace
       - Find and delete the WorkspaceMember record

    5. **changeMemberRole(UUID workspaceId, UUID userId, String newRole)**:
       - Find existing member record or throw RuntimeException
       - Only-admin guard: if removing admin role and user is the only admin, throw
       - Set role to newRole (uppercase), save

    6. **findWorkspacesByUserId(UUID userId)** — returns List<Workspace>:
       - Find all WorkspaceMember records for userId
       - Collect workspace IDs
       - Find all Workspace by those IDs where deleted_at IS NULL
       - Return the list

    7. **getUserRole(UUID workspaceId, UUID userId)** — returns Optional<String>:
       - Find WorkspaceMember by workspaceId + userId
       - Return member role or empty

    8. **findAllNonDeleted()** — returns List<Workspace>:
       - Replace workspaceRepository.findAll() with a query that filters out soft-deleted workspaces
       - Add `findAllByDeletedAtIsNull()` to WorkspaceRepository

    Add to `WorkspaceRepository.java`:
    - `List<Workspace> findAllByDeletedAtIsNull()`
    - `List<Workspace> findAllByIdIn(List<UUID> ids)` (if not already present)

    Add to `WorkspaceMemberRepository.java`:
    - `void deleteByIdWorkspaceIdAndIdUserId(UUID workspaceId, UUID userId)`
    - `long countByIdWorkspaceIdAndRole(UUID workspaceId, String role)`

    Follow existing code patterns: constructor injection, @Transactional on service methods, @Slf4j logging.
  </action>

  <name>Task 2: Create WorkspaceInvitationService with full lifecycle</name>
  <files>

  <action>
    Create `WorkspaceInvitationService.java` under `com.shiftleft.hub.workspace.service`:

    - @Service @RequiredArgsConstructor @Transactional @Slf4j
    - Inject: WorkspaceInvitationRepository, WorkspaceMemberRepository, WorkspaceRepository, UserRepository

    Methods:

    1. **sendInvitation(UUID workspaceId, UUID invitedUserId, UUID invitedByUserId, String role)**:
       - Validate workspace exists and is not soft-deleted
       - Validate user is not already a member (check WorkspaceMemberRepository)
       - Validate no PENDING invitation already exists for this user+workspace (D-09: no expiration, check existsBy...)
       - Build and save WorkspaceInvitation with status="PENDING"
       - Return saved entity

    2. **listByWorkspace(UUID workspaceId)** — returns List<WorkspaceInvitation>:
       - Find all invitations for workspace (both PENDING and non-PENDING for history)

    3. **listPendingForUser(UUID userId)** — returns List<WorkspaceInvitation>:
       - Find all invitations with status="PENDING" for this user

    4. **acceptInvitation(UUID invitationId, UUID userId)**:
       - Find invitation by id, throw if not found
       - Validate status == "PENDING", throw if already actioned
       - Validate invitedUserId == userId (can't accept someone else's invite)
       - Create WorkspaceMember with the invitation's role
       - Update invitation status to "ACCEPTED"
       - Log the acceptance

    5. **rejectInvitation(UUID invitationId, UUID userId)**:
       - Same validation as accept
       - Update status to "REJECTED"
       - No membership created

    6. **revokeInvitation(UUID invitationId, UUID workspaceId)**:
       - Find by id AND workspaceId (admin scoping)
       - Validate status == "PENDING"
       - Set status to "REJECTED" (or delete — use REJECTED for audit trail)

    Use helper methods to build WorkspaceInvitationResponse DTOs (with user email/displayName from UserRepository).
  </action>

    - Validates membership state before operations
